# IPNI Sub-Package

InterPlanetary Network Indexer for content discovery.

## Services

- **ipni-init** - One-time configuration initialization
- **ipni** - Content discovery indexer (storetheindex)

## Ports

| Port | Service | Description |
|------|---------|-------------|
| 3000 | ipni | Finder (queries) |
| 3002 | ipni | Admin |
| 3003 | ipni | Ingest (advertisement sync) |

## Configuration

- `config/ipni-config.json` - IPNI configuration (generated by ipni-init)

## Volumes

- `ipni-data` - IPNI persistent data and configuration

## Standalone Usage

```bash
cd systems/indexing/ipni
docker compose up -d
```

## Dependencies

None - this is a foundational service within the indexing system.

## Used By

- indexer (queries IPNI for content discovery)
